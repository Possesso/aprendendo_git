Aprendi que, basicamente, o Git é um sistema de controle de versão de arquivos (versionamento) muito utilizado em coisas como: desenvolvimento de software, registro de histórico, edição de arquivos e entre outros.
Um dos principais benefícios do Git, é o sistema distribuído, onde cada desenvolvedor pode ter um repositório autônomo e independente.
Pra início, os principais comandos do Git vão ser: Git add, Git commit, Git push, Git status, Git branch, Git checkout e Git init.
